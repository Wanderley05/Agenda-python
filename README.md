# 🚀 Cadastro Inteligente

Sistema de cadastro de clientes moderno desenvolvido com **HTML5, CSS3 e JavaScript**, com validação em tempo real, feedback visual e simulação de envio de dados.

---

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de aplicar conceitos de **Front-End moderno**, focando na experiência do usuário (UX) e boas práticas de desenvolvimento.

O sistema simula um formulário real utilizado por empresas para cadastro de clientes, garantindo validações eficientes e interface amigável.

---

## 🛠️ Tecnologias Utilizadas

- HTML5 (estrutura semântica)
- CSS3 (estilização moderna e responsiva)
- JavaScript (validação e interatividade)
- Git e GitHub (versionamento)

---

## ✨ Funcionalidades

### ✅ Formulário Completo
- Nome
- E-mail
- Senha
- Confirmação de senha

---

### ✅ Validação em Tempo Real
- Validação ao sair do campo (blur)
- Mensagens dinâmicas:
  - ✔️ **Válido (verde)**
  - ❌ **Inválido (vermelho)**
- Regras aplicadas:
  - Nome mínimo de 3 caracteres
  - Email com formato válido
  - Senha com:
    - mínimo 8 caracteres
    - 1 letra maiúscula
    - 1 número
  - Confirmação de senha obrigatória e igual

---

### 🔥 Indicador de Força da Senha
Barra dinâmica com 4 níveis:

- ❌ Fraca (vermelho)
- ⚠️ Razoável (laranja)
- ⚠️ Boa (amarelo)
- ✅ Forte (verde)

Critérios analisados:
- Tamanho mínimo
- Letra maiúscula
- Número
- Caractere especial

---

### 🚀 Simulação de Envio (Loading)
- Botão desabilitado ao enviar
- Texto alterado para **"Enviando..."**
- Spinner animado
- Simulação de API com `setTimeout`
- Mensagem final de sucesso
- Limpeza automática do formulário

---

### 🎨 Design e UX
- Layout centralizado
- Interface limpa e moderna
- Feedback visual com cores:
  - Verde = sucesso
  - Vermelho = erro
- Animações suaves (CSS transition)
